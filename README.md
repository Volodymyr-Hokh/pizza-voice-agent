# Pizza Voice Agent

Голосовий агент-помічник піцерії на **LiveKit Agents** та **OpenAI Realtime API**
(модель `gpt-realtime-mini`). Агент веде голосовий чат українською й допомагає клієнту
обрати страву, оформити та відстежити замовлення.

## Можливості

- Показати меню або окрему категорію — піца, напої, десерти.
- Розповісти деталі про страву: склад, ціну, наявність.
- Прийняти замовлення: зібрати позиції, ім'я, телефон та адресу.
- Перевірити статус замовлення за номером (напр. `ORD-101`).

Діалог природний: короткі усні репліки, без markdown.

## Стек

- [LiveKit Agents](https://docs.livekit.io/agents/) (Worker + `AgentSession`)
- OpenAI Realtime API — модель `gpt-realtime-mini`
- Python 3.10+

## Структура

| Файл | Призначення |
|------|-------------|
| `agent.py` | Точка входу: Worker, `AgentSession`, Realtime-модель, агент, привітання |
| `tools.py` | `@function_tool` обгортки, що підключають функції до агента |
| `fake_api.py` | Надані mock-дані та реалізація tools (без змін) |
| `requirements.txt` | Залежності |
| `.env-example` | Приклад змінних середовища |

## Встановлення

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1        # Windows PowerShell
# source .venv/bin/activate         # macOS / Linux

pip install -r requirements.txt

copy .env-example .env              # Windows
# cp .env-example .env              # macOS / Linux
```

Відкрийте `.env` і вставте свій `OPENAI_API_KEY`.

## Запуск

Агент працює локально через мікрофон і динаміки. Потрібен лише `OPENAI_API_KEY`.

```powershell
python agent.py console
```

Агент привітається голосом — можна одразу говорити.

## Приклади голосових сценаріїв

- «Покажіть меню» / «Які у вас є піци?» → агент перелічує позиції.
- «Розкажіть про Пепероні» → склад, ціна, наявність.
- «Хочу дві Маргарити, мене звати Іван, телефон..., адреса...» → агент підтверджує
  й оформлює замовлення, називає номер, суму та час доставки.
- «Який статус замовлення ORD-101?» → «Готується».

Тестові замовлення в mock-даних: `ORD-101` (Готується), `ORD-102` (Їде до вас).
